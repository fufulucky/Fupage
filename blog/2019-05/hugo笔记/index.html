利用hugo主题搭建个人主页并上传至github网站

需要做的前期准备

  • 下载好的hugo
  • Git工具
首先，选择好theme后，可以看它对应的使用指南，创建网站。
经常用到的指令：
            Hugo server %运行 
            Hugo new XXX.md  %XXX可以是中英文

接下来，将搭建好的网站上传至github

  • 第一步：在github中建立新的库。注意，命名必须为xxxxx.github.io
  • 第二步：在本地克隆这个库，使用指令git clone
  • 第三步：在本地库中，将所有的文件复制到本地库中，并更改本地库中的config文件,将baseURL改为发布后的网站
  • 第四步：在本地库中，git bash，依次输入以下指令:
            git init
            git remote add origin https://github.com/fufulucky/website.git  %存放源文件的
            git submodule add https://github.com/fufulucky/xxxxxx.github.io.git public %存放将来public里的数据 
            git add .
            git commit -m “Initial commit”
            git push -u origin master
            hugo  %将原始文件生成public 网站版本
            cd public
            git add .
            git commit -m “Build website”
            git push -u origin master
后续更新时，需要将.git文件夹里的内容删光，同时删除public文件夹。再次进行相同的操作。
